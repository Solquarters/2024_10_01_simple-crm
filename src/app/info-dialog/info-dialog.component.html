Welcome to My CRM (Customer Relationship Management) System

Navigation

  • Open the side navigation to access all parts.

Summary

  • Summarizes core data in a table.

Data Visualization

  • D3.js-based visualization of user data in relation to license count and residency population.
  • You can sort the data by license or city.
  • Hover or click on the circles to get detailed information.
  • Expand the legend to see customer cities and population data.

Customer List

  • Shows all customers in a table.
  • Push the + button to add a new customer.
  • Fill out the form and add licenses - a Base64 key is added to your license name.
  • Enter a valid city; the CRM fetches the current population data automatically through the WikiMedia API.
  • Click table rows for a detailed view where fields can be edited.

Real-Time Updates

  • Every change is visible in the table or the bubble chart due to firebase integration.
